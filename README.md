# Sales Performance & Customer Insights Analysis

## Project Overview
This project analyzes retail sales data to uncover trends in product categories, monthly sales, top-performing products, and regional performance.

## Tools & Technologies
- Python (Pandas, Matplotlib)
- Jupyter Notebook
- Power BI (to be added)

## Key Insights
- Technology category has the highest total sales
- September shows peak monthly sales
- West region generates the highest revenue
- A small number of products contribute disproportionately to total sales
